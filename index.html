The custom C++ code for the paper, 

Calcium-activated nonspecific cation current and synaptic depression promote network-dependent burst oscillations
Jonathan E. Rubin, John A. Hayes, Jeffrey L. Mendenhall, and Christopher A. Del Negro

is found here: NeuronetExperimenter
The README.txt file within the distribution is a quick start guide to getting the software built and running, while more extensive documentation, including tutorials and an installation guide, are found in the "docs" sub-directory as well as here.

Demos of the simulations in the paper are found here (along with other examples from ModelDB): NeuronetExperimenter-demos.tar.gz 

XPPAUT models for the same paper are found here: rhmd-xpp-models.tar.gz
